주제 결정

이미지 기반 고증 오류를 밝히는 모델로 결정

input : 이미지 + 연도

-> ai로 객체 찾아서 -> 데이터베이스 찾아서 연도 구분 -> 입력받은 연도와 비교

output : 얼마나 다른지

전체를 잡고 하기 보다는 기준 연도를 잡아서 하거나 특정 물품에 대해서 학습하는 것이 나을 것 같음
상대적으로 ai서비스 자체가 소셜미디어로 대중적으로 퍼짐
 -> 더이상 영상을 증거로 볼 수 없는 상황이 생김 / 영상의 문맥을 파악하거나 해서 ai모델이 만든 영상인지를 파악하는 방향으로도 갈 수 있을듯

굳이 얘기를 하자면 감지를 확실히 할 수 있는 측면으로 진행을 하는 것이 좋아보임. 사실과 얼마나 잘못된 평가를 내리는지 -> 성과를 낼 수 있지 않을까 (수치로 확실하게 보여주는 것이 좋음)

1. 주제를 잘 정했다
2. 주제와 과제를 잘 연결했다
3. 추가적인 주제를 결정해서 그거까지 완성하면 더 좋음
